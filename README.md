# Hip Sling Carrier Market Validation & Data Analytics Project

## 📋 Project Description

This project is a data-driven validation of a business concept for a premium hip sling carrier, targeting the EU market through an online-only sales model.

The project combines two goals in one workflow:
1. **Entrepreneurial validation** — determining whether this product concept is commercially viable before committing capital, factoring in EU regulatory requirements (EN 13209-2:2015, OEKO-TEX 100, REACH, CE marking) that are mandatory for selling children's products.
2. **Data analytics portfolio piece** — using the validation process as a real-world case study demonstrating survey design, data pipeline construction, statistical analysis, and dashboard reporting.

## 🎯 Motivation & Goal

New parents often face a trade-off between carriers that are comfortable for the wearer and ones that are affordable or stylish. This project asks a simple question with a data-backed answer: **is there a viable market for a premium, ergonomically improved hip sling carrier across the EU, and does the business model hold up once real regulatory and production costs are factored in?**

The end goal is twofold:
- A clear **go / no-go business recommendation**, backed by survey data, willingness-to-pay analysis, and financial modeling (with and without EU certification).
- A **polished, publicly shareable analytical case study** — from raw survey data to a Looker Studio dashboard — that demonstrates end-to-end data analytics skills (Google Forms → BigQuery → Python/Colab → Looker Studio).
